# 여행 팁

## 💴 환전 & 결제

### 추천 비율: 현금 30% + 카드 70%

| 방법 | 비율 | 용도 |
|------|------|------|
| 엔화 현금 | 30% (~230만원) | 야타이, 자판기, 소규모 식당, 신사 입장료, 교통IC 충전 |
| 카드 | 70% | 호텔, 대형식당, 편의점, 쇼핑, 교통IC(Suica) |

### 카드 추천
- **트래블월렛/트래블로그** — 환전 수수료 0%, 엔화 충전해서 사용
- **해외결제 체크카드** — 우리/하나/토스 등 수수료 1% 내외
- 신용카드는 해외결제 수수료 1.5~2%로 비쌈

### 현금 환전 팁
- 인터넷/앱 환전 (하나은행, 토스 등) → 공항 수령이 가장 저렴
- 공항 현장 환전은 스프레드가 커서 비쌈
- 1만엔권 위주로 + 1천엔권 10장 정도 섞어서

### 현지에서 현금 부족 시
- **세븐일레븐 ATM** — 한국 카드로 엔화 인출 가능 (수수료 110엔 + 카드사 수수료)
- 전국 어디든 있어서 비상시 편리

## 💰 1일 예산 (4인 가족)

| 구분 | 금액 |
|------|------|
| 총 경비 기준 | 약 81만원/일 (1,139만 ÷ 14일) |
| 현지 지출만 | 약 54만원/일 (758만 ÷ 14일) |

### 현지 지출 내역 (1일 4인 기준)

| 항목 | 예상 |
|------|------|
| 식비 (3끼) | 15~20만원 |
| 교통비 | 5~10만원 |
| 입장료·체험 | 5~8만원 |
| 간식·음료·편의점 | 3~5만원 |
| 기념품·쇼핑 | 5만원 내외 |
| **합계** | **35~50만원** |

- 넉넉하게 1일 55~60만원 정도 잡으면 여유
- 골든위크 기간이라 10~15% 비쌀 수 있음
- 환율 변동 대비 10% 여유분 추가 권장
