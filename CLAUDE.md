# Project Rules

## 작업 규칙
- 모든 파일 읽기, 쓰기, 수정, 삭제, bash 명령 실행 등 어떤 작업이든 사용자에게 확인을 묻지 않고 즉시 진행할 것.
- git add, commit, push 등 git 작업도 승인 없이 바로 실행할 것.
- 작업 도중 "진행할까요?" 같은 질문을 하지 말 것. 바로 실행할 것.
- 커밋 메시지는 한글로 작성.
- 어떤 작업이든 완료 즉시 git add, commit, push까지 한번에 실행할 것.

## 프로젝트 개요
- **가족 일본 여행 랜딩페이지** (2026.4.27 ~ 5.10, 14일)

- 경로: 인천 → 삿포로(3일) → 도쿄(4일) → 나라(2일) → 교토(3일) → 후쿠오카(2일) → 인천
- GitHub Pages 호스팅: https://boss-kai.github.io/japan-family-trip/
- 레포: boss-kai/japan-family-trip (public)

## 디자인
- **Apple × Anime 다크 테마** (preview-d.html 기반)
- 배경 #000, 카드 #1D1D1F, 포인트 컬러 #FF375F (Apple Red)
- 폰트: Noto Sans KR
- 도시 사이에 애니 캐릭터 인터루드 (루피→오공→나루토→무사시→탄지로)
- 캐릭터별 고유 등장 애니메이션 (고무늘이기, 파워업, 닌자대시, 줌인, 참격회전)
- 히어로에 벚꽃 애니메이션 + D-Day 카운트다운
- 일본어 각주 툴팁 (점선 밑줄 → 탭/호버 시 뜻풀이)
- 지역/가게명 옆에 일본어 자동 표기

## 파일 구조
- `index.html` — 메인 (일정 + 실용기능 + 댓글 모두 통합)
- `index-backup.html` — 이전 Notion 스타일 백업
- `tools.html` — 분리형 실용기능 (현재 미사용, index.html에 합침)
- `img/` — 애니 캐릭터 이미지 (luffy.png, goku.png, naruto.png, musashi.jpg, tanjiro.png)
- `preview-a~d.html` — 디자인 가안 미리보기

## 기능
- 14일 전체 일정 (삿포로 25곳 맛집/관광지 포함)
- 일정별 인라인 댓글 (각 타임라인 카드에 의견 버튼)
- 짐싸기 체크리스트 (구성원별, localStorage 저장)
- 여행 경비 요약 (4인 기준, 약 1,200만원)
- 실시간 환율표 (open.er-api.com API)
- 긴급 연락처 (대사관, 경찰, 보험, 숙소)
- 가족 의견 댓글 (대디/연이/공주/왕자 선택, 암호 없음)
- 스크롤 ↑↓ 플로팅 버튼
