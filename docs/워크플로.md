# 스케줄 확정 & 비용 산정 워크플로

## 1단계: 스케줄 확정

**현재 상태**: 대략적인 일정은 있지만, 실제 운영시간/가격 미확인

**해야 할 것**:
- 각 장소의 **운영시간, 휴무일** 확인 (골든위크 특별 영업 포함)
- **예약 필수 여부** 확인 (팀랩, 코카콜라 공장 등)
- 동선 최적화 (구글맵으로 실제 이동시간 체크)

**방법**: Claude가 각 장소를 웹 검색해서 확인. 도시별로 하루씩 진행.

## 2단계: 비용 정밀화

**현재 상태**: data-jpy가 추정치

**해야 할 것**:
- **식당**: 실제 메뉴 가격 검색 → 4인(6인분) 기준 산정
- **입장료**: 공식 사이트에서 어른/어린이 요금 확인
- **교통비**: 구간별 실제 요금 (IC카드/현금 차이)

**방법**: 각 장소 공식 사이트나 tabelog/Google Maps에서 가격 확인 → data-jpy 업데이트 → 경비 자동 반영

## 3단계: 예약

확정된 스케줄 기반으로:
- [ ] 호텔 예약 (Booking.com/Airbnb)
- [ ] 삿포로→도쿄 항공편 (스카이스캐너)
- [ ] 사전예약 필수 장소 예약
- [ ] 신칸센 (출발 1개월 전부터 예약 가능)

## 진행 방식

도시 하나씩 순서대로:

```
"삿포로 3일 검증해줘"
→ 웹 검색으로 운영시간/가격 확인
→ data-jpy 업데이트
→ 경비 자동 반영
→ 다음 도시
```
